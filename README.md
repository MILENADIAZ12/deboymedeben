Esta es una app de recordatorio para personas que necesitan anotar quiénes les debe dinero y a quién le deben. Cuenta con inicio de sesión y registro.
Permite agregar contactos filtrándolos desde la agenda del celular y añadir una descripción, el monto y la fecha límite. Cada registro se agrega a la pantalla principal.
La app también envía notificaciones a las 8 de la noche, un día antes de que venza la fecha límite.
En el menú se encuentran las opciones de Home, Perfil y Calendario. El calendario marca las fechas que se añadieron,
el home es la pantalla principal que se visuliza las deudas y el pefil es para cerrar sesión.
